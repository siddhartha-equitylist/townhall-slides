# Townhall Slides

Static HTML slide deck for the June 2026 EquityList town hall.

Vercel serves `index.html` at the root URL. `June_2026_Townhall.html` is kept as
the named source copy of the deck.

## Deploy on Vercel

Use these settings when importing the repository:

- Framework Preset: Other
- Build Command: None
- Output Directory: `.`
- Install Command: None
